**Marca Comercial:** Art-tech  
**Stack Tecnológica:** Linux (WSL 2), PostgreSQL, Python, Git/GitHub  

## 🚀 O Projeto
A **Art-tech** nasceu da união entre a sensibilidade do artesanato estruturado (bolsas em telas e crochê) e o rigor técnico da Engenharia de Dados. O objetivo é criar um ecossistema de gestão comercial, controle de estoque, análise de insumos e precificação automatizada baseada em dados reais de mercado.

## 📊 Arquitetura do Sistema
- **Banco de Dados (PostgreSQL):** Modelagem relacional para gerenciar fornecedores, insumos, ferramentas de produção, controle de tempo e vendas.
- **Análise & Automação (Python):** Scripts para cálculo automático de margem de lucro, ponto de equilíbrio (break-even) e insights de tendências de mercado.
- **Infraestrutura:** Ambiente puríssimo rodando em ambiente Linux via WSL 2.
